## unix-cat

unix-cat == console.log<br>
cat() is less work than console.log()<br>
this one is for the lazy cats

### installation

    npm install unix-cat

### usage
```javascript
cat('hello world'); // hello world
cat([1,2,3]);       // [1,2,3]
cat({key:'value'}); // {key:'value'}
cat(12345);         // 12345
cat(1.2345);        // 1.2345
```
